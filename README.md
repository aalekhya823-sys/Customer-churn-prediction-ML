# Customer Churn Prediction ML Project

This project predicts whether a telecom customer will churn using Machine Learning.

## Technologies
Python
Scikit-learn
Streamlit
Pandas
Seaborn

## Model
Random Forest Classifier

## Features
- Data preprocessing
- Feature encoding
- Model training
- Streamlit web app

## How to Run

pip install -r requirements.txt
streamlit run app.py
